#Rascunho

LOAD BALANCE COM SITE ARMAZENADO NO APACHE

INTRODUÇÃO

Nesse labrotario será feito um load balance com um site. Esse site se encontra armazenado em um servidor Apache na VM Ubuntu. Será criado uma nova VM Ubuntu clonando as mesmas configurações da VM já existente. Será criado também uma VM que utilizara o Nginx para fazer o load balance e outra vm que será o cliente parafazer os testes e validações da balanceamento feito pelo Nginx.  


Antes de quais quer ações, foi feito a subida das VM's para o ambiente do virtualbox.
